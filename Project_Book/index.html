Project－Book可或缺圖書志工系統

Apr 28, 2017

Book可或缺圖書志工系統

專題成果

專題小組共有3人，題目為「Book或缺圖書志工系統」，以臺中市大墩文化中心之圖書館為例，根據目前的志工系統做改良，並加入許多功能，為一動態網頁，使用PHP + CSS + jQuery + JavaScript + MySQL建置，本人負責資料搜集、系統分析與設計、資料庫正規化與建置、系統功能轉寫及撰寫系統文件。

困難與成長

在做這個專題時，這是第一次完完全全重頭到尾都是團隊與自己一同寫出來，且是透過沒有學過的語言開始著手，初期在學習PHP和伺服器的架構上遇到了一些瓶頸，但透過多次學習和嘗試後慢慢的上手，到後期在設計的流程上和內容的完整上加上更多的細節，讓系統更加完整，也更加符合實際應用。最終，也在2016資應小專中獲得第一名的肯定。
